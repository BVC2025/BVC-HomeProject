# Appendix A — Glossary

| Term | Meaning |
|---|---|
| **AMC** | Annual Maintenance Contract — post-sale service agreement (planned Phase 7). |
| **BOM** | Bill of Materials — list of materials and processes that go into one product unit. |
| **BVC** | Bharath Vending Corporation — the customer using this ERP. |
| **BVC24** | The product / system name (vendor brand). |
| **CallMeBot** | Free third-party WhatsApp bridge used as a no-cost fallback channel. |
| **CRM** | Customer Relationship Management — the customer + enquiry + requirement domain. |
| **DRAFT** | Editable state before a document is sent or committed. |
| **ERP** | Enterprise Resource Planning — the category of system this is. |
| **Gantt** | The day-by-day work-order timeline view for a Work Order. |
| **GRN** | Goods Receipt Note — supplier delivery record against a PO. May be partial. |
| **GST** | Goods & Services Tax — Indian indirect tax (18 % default). |
| **HSN** | Harmonized System of Nomenclature — Indian product classification code. |
| **IST** | India Standard Time (UTC+05:30). |
| **JWT** | JSON Web Token — the authentication token format used. |
| **LOP** | Loss of Pay — unpaid leave, auto-applied when paid quota is exhausted. |
| **MD** | Managing Director — top recipient of WhatsApp alerts and the MD Review screen. |
| **NCR** | Non-Conformance Report — a QC failure that must be resolved. |
| **PO** | Purchase Order — order issued to a supplier. |
| **PROCESS** | A BOM item type indicating in-house manufacture (vs. PURCHASE). |
| **PURCHASE** | A BOM item type indicating external sourcing from a supplier. |
| **QC** | Quality Control — inspections and NCRs. |
| **Resend** | Third-party transactional email service (HTTP API), preferred over SMTP. |
| **SO** | Sales Order — binding contract following an approved Quotation. |
| **STAR** | The monthly performance rating system (1–5 stars, weighted formula). |
| **SU** | Sub-Project (template) — a project structure template. |
| **TLS** | Transport Layer Security — encryption for HTTPS and SMTP. |
| **UPI** | Unified Payments Interface — Indian instant payment system, used in BVC machines. |
| **VENDOR** | Tenant of the ERP (currently BVC; the model supports multi-tenant). |
| **WO** | Work Order — a production run. |
| **WO Stage Progress** | Per-WO per-stage row tracking who is doing what, when. |
